4. Conda常用指令：
	#1. 查看信息：conda info
	#2. 查看环境：conda info -e 或者conda env list
	#3. 查看源：
	#4. 新建环境：conda create -n ic-kg python==3.7
	#5. 删除环境：conda remove --name ic-kg --all
	#6. 激活环境：conda activate ic-kg
	#7. 退出环境：conda deactivate
	#8. 切换环境：conda activate <其他环境>
	#9. 查询当前环境安装包：conda list
	#10. 搜索包：conda search PIL
	#11. 安装包：conda install PIL
	#12. 升级包：conda update python
	#13. 卸载包：conda uninstall pandas
